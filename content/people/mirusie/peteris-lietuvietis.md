# Pēteris Lietuvietis (Kamaldiņš)

{{< callout type="info" >}}
**Statuss:** Dzimtas loceklis
{{< /callout >}}

**Pēteris Lietuvietis** (arī **Kamaldiņš**) bija Kamaldiņu dzimtas pārstāvis.

| Dati | Informācija |
|------|-------------|
| **Dzimšanas datums** | 1850. gads |
| **Dzimšanas vieta** | Rīgas Jēzus ev-lut draudze |
| **Miršanas datums** | Nezināms |
| **Tēvs** | Krišs Kamaldiņš |
| **Māte** | Mīne Kamaldiņa |
| **Sieva** | Lote Lietuviete |
| **Brāļi/Māsas** | Jēkabs, Otto, Kārlis, Jānis un citi |
